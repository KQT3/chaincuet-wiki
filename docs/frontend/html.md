## Html For Redability

- **Headings:** h1, h2, h3, h4, h5, h6
- **Paragraphs:** p
- **Lists:** ul, ol, li
- **Tables**: table, thead, tbody, tr, th, td
- **Semantic elements:** header, nav, main, section, aside, footer
- **Form elements:** form, input, textarea, select, button
- **Images:** img
- **Links:** a
